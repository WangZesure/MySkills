# Iteration Log

> Append-only. Each iteration adds one version entry.
> For iteration-driven or hybrid tasks.

## Format

```
### [YYYY-MM-DD] V[N]: [Title]
- **What Changed**: [Concrete details — files, parameters, decisions]
- **Current State**: [Status after this iteration]
- **Feedback**: [User feedback or test results that triggered this round]
- **Next**: [What to solve next]
```

---

## Entries

### [YYYY-MM-DD] V1: Initial
- **What Changed**: ...
- **Current State**: ...
- **Feedback**: N/A (initial)
- **Next**: ...
